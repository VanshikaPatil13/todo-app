📝 Todo App

This is a simple Todo app made using JavaScript and Object-Oriented Programming (OOP).

You can add, edit, and delete tasks. All your tasks are saved in the browser using localStorage, so they won’t disappear when you refresh the page.


🚀 What this app does
👉 Add new tasks
👉 Edit tasks
👉 Delete tasks
👉 Saves tasks in browser automatically
👉 Shows your tasks even after refresh

🧠 How it is built

This project uses a JavaScript class called TodoApp.

The class handles everything like:

* adding tasks
* deleting tasks
* editing tasks
* showing tasks on screen
* saving data in browser

📁 Files
index.html → structure of the page
style.css → styling
script.js → all the logic (OOP class)

▶️ How to use
1. Open the project in browser
2. Type a task in the input box
3. Click Add
4. Your task will appear in the list
5. You can edit or delete anytim5

* How to work with DOM
* How to store data in localStorage
